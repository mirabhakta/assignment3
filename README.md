## Setup
1. Clone this repo.
2. Open 'config.properties' and add your API Key and mode, e.g. 'GEMINI_API_KEY=your-key-here' and 'GEMINI_MODEL-gemini-2.5-flash'
3. Build and run the application from your IDE (or with Maven/Gradle if configured).

## Features
- Creative Writing Mode
- Professional Writing Mode
- Academic Writing Mode
- Easy File Input System
